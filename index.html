Machine Learning as a service

  • It is IOTA tangle
  • Any IOTA node can be turned into a machine learning node
  • Machine Learning nodes create CognIOTA cluster.
  • IOTA nodes request machine learning services from CognIOTA.
  • CognIOTA finds the solution for the request.
  • It uses auctions for finding the best task executer.
  • CognIOTA powers
    the economy of IoT.

    All participants are connected to cognIOTA.

  • Stage I: Shop's initial Request

    When a shop needs to replenish its shelves, it sends a request to cognIOTA.

  • Stage II: Warehouses goods auction

    CognIOTA conducts auction to find the warehouse with the best offerings, such as price, location, reputation. Finally, asks IOTA to sign contract.

  • Stage III: Cars delivery Auction

    CognIOTA conducts auction to find the self-driving cars with the best offerings, such as price, delivery time, reputation. Finally, asks IOTA to sign contract.

  • Stage IV: Delivery escort service

    The car uses cognIOTA's machine learning service to find the optimal delivery route. CognIOTA provides on the route escort service to react at the constantly changing environment.

  • Stage V: Payment

    When all contracts conditions are met. Warehouse, car, and cognIOTA are paid for their services with IOTAs.
